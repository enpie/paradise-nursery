# Paradise Nursery - Houseplant Shopping Application

An e-commerce application built with React and Redux Toolkit for buying beautiful houseplants.

## Features
- Landing page with a "Get Started" button.
- Product listing page with 3 categories and 6 unique plants per category.
- Add to cart functionality (disables button after adding).
- Dynamic shopping cart icon with real-time item counter.
- Cart page with functionality to increase, decrease, or delete items.
- Real-time order total calculation.
